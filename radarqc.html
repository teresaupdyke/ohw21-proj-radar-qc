class radarqc.dataset.DataSet(paths, preprocess)¶

Bases: object

Supports aggregation of all Cross-Spectrum files in a given directory into a batch of images.

Uses the monopole antenna channel (Antenna 3) for the spectrum

Returns an iterable containing the Cross-Spectrum file header for each input path

Return type

property spectra¶
